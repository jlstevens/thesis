All the code necessary to re-run all the simulations and regenerate all the figures may be found in the publications repository. In particular, there are two notebooks that reproduce all the figures used in the paper which form a subset of the figures used in this chapter of the thesis:

  • Model definition : A notebook that simultaneously defines the model and explores it. Running the live version of this notebook generates the Topographica model file 'gcal.ty'.

  • Simulation and Figures : Defines all 842 model simulations needed to reproduce all the figures, allows you to launch them and automatically builds the corresponding SVG figures used in publication. Although running the full set of simulations is computationally expensive, the live notebook allows you to select a small subset before launching the jobs.

Figure 5.1¶

Figure 5.1 : Polar power spectrum analysis of a range of experimental and artificial orientation maps.

This figure runs the polar power spectrum analysis for four different conditions (1) Experimental data derived from chronic ferret recordings, published in Chapman et al. 1996 (2) a GCAL model simulation with a cortical area of 3 (3) the central unit area of the same simulation (4) a uniformly random orientation preference map.

Figure 5.1A (Experimental data)¶

The following cell applies polar power spectrum analysis to the experimental data, reproducing Figure 5.1A. Note that the warning regarding the contour identification failure can simply be ignored as this figure does not make use of pinwheel locations.

In [11]:
from stevens_thesis16 import asset_path
rgb = hv.RGB.load_image(asset_path+'/chapman/chapman.png')
fft_im = hv.RGB.load_image(asset_path+'/chapman/chapman_fft.png').data

fft_slice = np.array(fft_im)[1:168,1:168,0]
fft_slice -= fft_slice.min()
fft_slice = fft_slice.astype(float)
fft_slice /= fft_slice.max()

window = 2
slc = slice(167/2-window,167/2+window-1)
fft_slice[slc,slc] = 1

fft_power = hv.Image(fft_slice).relabel(group='FFT Power')
noisy = hv.Image(np.zeros((167,167)) * np.pi, group='Orientation Preference',
        vdims=[hv.Dimension('Orientation Preference', cyclic=True, range=(0,np.pi))])
analysis = PowerSpectrumAnalysis(noisy * fft_power)
rgb + analysis.PowerFit + fft_power

Figure 5.1B (GCAL, cortical area=3.0)¶

The following cell applies polar power spectrum analysis to the GCAL model using a cortical sheet area of $3.0$, reproducing Figure 5.1B. To run the model, we need to use the topographica neural simulator and import ModelGCAL:

In [12]:
import topo
from topo.submodel.gcal import ModelGCAL

Next we run the model for 20000 iterations before measuring the orientation preference and selectivity. The same polar power spectrum analysis is then applied. Note that this cell can take a long time to run as a large number of iterations are being executed for a large simulation:
